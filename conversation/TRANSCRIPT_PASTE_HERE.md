This file is intentionally a placeholder. Claude cannot export a guaranteed
word-for-word transcript of the whole conversation (early parts were compacted),
so the **authoritative verbatim record must come from you**, exported from the
Claude interface. The structured reconstruction lives in
`genesis_journey_full_record.md`; this file is for the raw, exact text.

## How to get the verbatim transcript (do this once)

**Option A — copy from the web app (simplest):**
1. Open this conversation in claude.ai on your computer.
2. Scroll to the very top of the conversation.
3. Select all the text from the first message to the last (click at the top,
   shift-click at the bottom, or Ctrl/Cmd-A inside the conversation area).
4. Copy, then paste it **below the line** in this file.

**Option B — official export (most complete, includes everything):**
1. In Claude settings, look for **Privacy / Data → Export data** (account-level
   export). It produces a file (often JSON) with your full conversation history.
2. Save that export into this `conversation/` folder as e.g.
   `claude_export.json`, and commit it. (You can keep the raw export *and* paste
   a readable version below.)

**Tip:** Once pasted/added, commit it so the verbatim record is timestamped in
git alongside everything else — that timestamp is part of your originality
protection (see `PROTECTION.md`).
